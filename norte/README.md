# AWS Community Day Norte 2026

Site oficial do AWS Community Day Norte 2026 — a maior conferência de cloud da Amazônia.

## Stack

- Next.js 16 (static export)
- Tailwind CSS v3
- shadcn/ui + Radix UI
- TypeScript
- Nextjs Studio

## Desenvolvimento

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
```

O site estático é gerado em `dist/`.
